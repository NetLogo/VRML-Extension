VRML Extension

The VRML extension allows you to build up 3D scenes and export them in VRML (or X3D) file format. Currently it only provides basic support for a few 3D objects, but it could be extended to do more. The VRML extension can be used both in Netlogo 2D and Netlogo 3D, although it is generally more effective when using NetLogo 3D.

How to use

To use the VRML Extension in your model, add a line to the top of your procedures tab: 

extensions [vrml]

VRML extension Primitives

vrml:clear-scene vrml:set-color vrml:add-sphere vrml:add-box vrml:add-cylinder vrml:save-scene vrml:save-scene-x3d

Details

vrml:clear-scene

vrml:clear-scene

clears the invisible scene that we are building up.

vrml:set-color

vrml:set-color R G B

Sets the active paint color to the color with the specified R G B component values (each component is between 0 and 255). All shapes added to the scene after this call will be this color (until vrml:set-color is called again).

vrml:add-sphere

vrml:add-sphere X Y Z radius

adds a sphere of radius radius to the scene at the coordinates (X,Y,Z). 

        example: ask turtles [ vrml:add-sphere xcor ycor zcor (size / 2) ]

vrml:add-box

vrml:add-box X Y Z X-width Y-width Z-width

adds a rectangular prism to the scene centered at the coordinates (X,Y,Z), with the given dimensions (X-width, Y-width, Z-width). 

        example: ask turtles [ vrml:add-box xcor ycor zcor 3 0.5 0.5 ]

vrml:add-cylinder

vrml:add-cylinder X1 Y1 Z1 X2 Y2 Z2 radius

adds a cylinder to the scene, with the axis of the cylinder running from the coordinates (X1,Y1,Z1) to (X2,Y2,Z2), and with a given radius radius. 

example: ask links [ 
           vrml:add-cylinder [xcor] of end1 [ycor] of end1 [zcor] of end1
                             [xcor] of end2 [ycor] of end2 [zcor] of end2
                             0.5 
         ]

vrml:save-scene

vrml:save-scene filename

Saves/exports the currently built scene to a VRML file format, named filename. (Highly recommended that you give filename a ".wrl" extension, so that VRML viewers recognize it.) 

        example: vrml:save-scene "MyCrazyShape.wrl"

vrml:save-scene-x3d

vrml:save-scene-x3d filename

Saves/exports the currently built scene to an X3D file format, named filename. 

        example: vrml:save-scene "MyCrazyShape.x3d"
